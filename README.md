# mx-master-3
My mouse configuration for Ubuntu.

# important!!
Disable the Scroll Lock when using the ABNT2 keyboard.

* Steps:
1. open this file: sudo nano /usr/share/X11/xkb/symbols/br
2. comment the line: "modifier_map Mod3   { Scroll_Lock };"
3. save and reboot.
